UnblockNeteaseMusic(三):伸手党使用教程

发表于 2020-06-24 | 更新于: 2020-06-30 | 分类于 其他
字数统计: 464 | 阅读时长 ≈ 1

前言：

此篇作为伸手党使用教程，即嫌麻烦不想搭建的人（虽然搭建也不麻烦）。
UnblockNeteaseMusic(一):Windows搭建教程 ，UnblockNeteaseMusic(二):Linux搭建教程

目的：

通过此项目可实现无限制收听网易云音乐（VIP，无版权都可听）

原理：

使用 QQ / 虾米 / 百度 / 酷狗 / 酷我 / 咪咕 / JOOX 音源替换变灰歌曲链接 (默认仅启用一、五、六)。
为请求增加 X-Real-IP 参数解锁海外限制，支持指定网易云服务器 IP，支持设置上游 HTTP / HTTPS 代理。
完整的流量代理功能 (HTTP / HTTPS)，可直接作为系统代理 (同时支持 PAC)。

使用：

PAC地址：http://47.93.103.89:4567/proxy.pac

Windows：设置 > 工具 > 自定义代理 (客户端内) > HTTP代理 > 服务器: … > 端口: …

Android：WLAN > 修改网络 > 高级选项 > 代理 > 代理自动配置 > PAC网址: …

macOS：系统偏好设置 > 网络 > 高级 > 代理 > 自动代理配置 > URL: …

  1. 在 Mac 上，选取苹果菜单 “系统偏好设置”，然后点按“网络”。
  2. 在列表中选择网络服务（例如，以太网或 Wi-Fi）。
  3. 点按“高级”，然后点按“代理”。
  4. 如果您要自动配置代理服务器设置，请选择“自动发现代理”以自动发现代理服务器，如果您使用的是自动代理配置 (PAC) 文件，则请选择“自动代理配置”。如果您选择“自动代理配置”，则请在 URL 栏输入 PAC 文件的地址。如果需要更多信息，请联系网络管理员。

IOS：无线局域网 > HTTP 代理 > 配置代理 > 自动 > URL: …

  1. 打开设置, 选择Wi-Fi
  2. 选择当前使用的热点
  3. 拖到最下面的代理设置，选择”自动”，填写PAC地址
